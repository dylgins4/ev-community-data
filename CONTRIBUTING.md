# Contributing

## The Easy Way (No Git Required)

**Option A:** Go to [Issues](../../issues/new), attach your rescue sheet PDF, and tell us the make and model. Done!

**Option B:** Upload directly to the [`submissions/`](submissions/) folder using GitHub's web interface — [click here to upload](../../upload/main/submissions).

## The Git Way (If You're Comfortable With It)

1. Fork this repo
2. Add a folder under `submissions/` named `MAKE_MODEL` (e.g., `BYD_SEAL`)
3. Drop the rescue sheet PDF in there
4. Add a `metadata.yaml` — copy from [`submissions/_template/metadata.yaml`](submissions/_template/metadata.yaml)
5. Open a pull request

## What Happens Next

1. We review the PDF for accuracy and source legitimacy
2. We process it into the app's knowledge base
3. We push it live — the vehicle appears in the app for all users

Reviews typically take a few days. We may ask questions if anything is unclear.
